### O projeto

Os usuários deverão:

- Digitar um número para verificar se o mesmo é primo ou não;
- Logo em seguida, clicar em PESQUISAR para visualizar o resultado;
- Os usuários poderão pesquisar qualquer número e quantas vezes for necessário.
